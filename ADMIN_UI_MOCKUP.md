## UI Components Used

- **Section Container**: Purple gradient background (`bg-gradient-to-r from-purple-500/5 to-pink-500/5`)
- **Button**: Primary button with Activity icon and Loader2 for loading state
- **Badge**: Success (green) or Destructive (red) variants for pass/fail counts
- **Test Cards**: 
  - Green border/background for PASS (`bg-green-500/5 border-green-500/30`)
  - Red border/background for FAIL (`bg-red-500/5 border-red-500/30`)
- **Icons**: CheckCircle2 (green) for pass, XCircle (red) for fail
- **Details**: Collapsible `<details>` element with JSON formatting
- **Animation**: Framer Motion AnimatePresence for smooth expand/collapse
