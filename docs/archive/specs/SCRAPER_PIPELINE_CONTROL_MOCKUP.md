## Interactive States

### Hover
- Buttons: Slight opacity change and scale
- Cards: Subtle shadow increase

### Disabled
- Gray background
- Reduced opacity
- No pointer cursor

### Loading
- Spinner animation replaces icon
- Text changes to "Running..." / "Queueing..." / "Retrying..."
- Button disabled

### Success
- Toast notification from top-right
- Green checkmark icon
- Auto-dismiss after 5 seconds

### Error
- Toast notification from top-right
- Red X icon
- Requires manual dismiss
